# 📖 Overview

AEGIS (Automated Educational Guide with Intelligent System) is an educational quadruped robot designed to bring artificial intelligence and robotics closer to students through interactive learning experiences.

Its modular architecture combines computer vision, intelligent locomotion, Bluetooth communication and educational activities to promote STEM learning and environmental awareness.

---

# ✨ Features

- 🤖 Quadruped robot
- 👁️ HuskyLens AI Vision
- 📱 Bluetooth communication
- 🎵 DFPlayer Mini audio
- ⚙️ 12 Servo Motors
- 🔋 Battery powered
- 🖨️ Fully 3D Printed
- 🌱 Environmental education

---

# 🎯 Educational Modes

## 👤 Face Recognition

Recognizes previously learned faces and interacts with the user.

---

## 🎨 English Color Learning

Identifies colors while teaching their pronunciation in English.

---

## ♻️ Waste Classification

Recognizes recyclable materials to promote environmental awareness.

# 🛠 Hardware

| Component | Model |
|-----------|--------|
| Microcontroller | ESP32 DevKitC V4 |
| Vision | HuskyLens 2 |
| PWM Driver | PCA9685 |
| Audio | DFPlayer Mini |
| Servos | 12x Servo Motors |
| Batteries | Li-Ion 18650 |
| Structure | PLA+ / PETG |

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/chicaizadiego55-tech/AEGIS-V1.0.git
```

Open the firmware

```text
Firmware/AEGIS.ino
```

Compile using Arduino IDE with ESP32 Boards installed.
