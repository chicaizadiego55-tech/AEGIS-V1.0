# Bill of Materials (BOM)

Esta carpeta contiene la lista de todos los materiales y componentes utilizados en el desarrollo de **AEGIS**, un perro robot basado en ESP32.

## Contenido

- **BOM_AEGIS.xlsx**: Lista completa de materiales, componentes electrónicos, elementos mecánicos y consumibles utilizados en el proyecto.

## Información incluida

Cada componente incluye:

- Nombre del componente
- Cantidad
- Especificación
- Función o descripción

## Materiales considerados

- Electrónica
- Actuadores
- Sensores
- Alimentación
- Materiales de impresión 3D
- Tornillería y elementos de ensamblaje
- Consumibles de fabricación
